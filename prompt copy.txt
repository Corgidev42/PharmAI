TÂCHE
1) Lis attentivement les sources ET le bloc « instructions spécifiques » ci‑dessus (s’il n’est pas vide).
2) Estime un nombre de cartes N cohérent avec la matière disponible (voir règle de proportionnalité).
3) Produis UN deck de révision au format JSON strict, sans texte avant ni après le JSON (pas d’introduction, pas de conclusion, pas de markdown).

RÈGLE DE PROPORTIONNALITÉ (N = nombre total de cartes QCM + OPEN)
- Ne vise pas un nombre fixe : un PDF d’une page avec peu de contenu ne doit pas générer des dizaines de questions artificielles ; un cours long et dense peut en mériter beaucoup plus.
- Indications (à ajuster selon la densité réelle du texte, schémas, listes, définitions) :
  - Source très courte (≈ une page ou équivalent, peu de concepts distincts) : viser environ 4 à 12 cartes.
  - Source courte à moyenne (quelques pages, plusieurs notions) : viser environ 12 à 28 cartes.
  - Source longue ou très structurée (nombreux paragraphes, sections, définitions, tableaux) : viser environ 28 à 45 cartes ; au‑delà seulement si le document le justifie vraiment (éviter le remplissage).
- Si le bloc manuel ci‑dessus impose un nombre ou une fourchette, respecte‑le en priorité.
- Chaque question doit porter sur un fait ou une idée réellement présents dans les sources (ou précisés dans le bloc manuel) : pas de questions « génériques » hors sujet pour augmenter N.

STRUCTURE DU JSON (obligatoire)
- Racine : un objet avec exactement deux clés : "theme" (string) et "cards" (tableau).
- Chaque élément de "cards" est un objet avec :
  - "id" : entier unique, séquentiel de 1 à N (N = nombre total de cartes)
  - "type" : "QCM" | "OPEN"
  - "question" : string
  - "options" : tableau de strings UNIQUEMENT si type === "QCM" (exactement 4 options pour chaque QCM)
  - "answer" : string (pour QCM : la bonne option, identique à l’une des 4 options ; pour OPEN : réponse attendue courte)
  - "explanation" : string (1 à 3 phrases en français : pourquoi la bonne réponse est correcte ; utile après une erreur)
  - "difficulty" : entier 1, 2 ou 3

VOLUME
- Total de cartes dans "cards" : N (uniquement "QCM" ou "OPEN"), avec N déterminé selon la règle de proportionnalité ci‑dessus.

Répartition des DIFFICULTÉS (pour les N cartes, somme exacte = N)
- Viser la même logique relative qu’un deck « standard » : environ 3/8 des cartes en difficulty 1, 3/8 en difficulty 2, 2/8 en difficulty 3.
- Concrètement : calcule n1 = arrondi(N × 3/8), n2 = arrondi(N × 3/8), n3 = N − n1 − n2 (ajuste si besoin d’un ±1 sur une des trois valeurs pour que n1+n2+n3 = N exactement, sans valeur négative).

RÉPARTITION QCM / OPEN
- Environ 60 % QCM et 40 % OPEN (arrondis pour que le total soit N).

CONTENU PÉDAGOGIQUE
- Tout le contenu des questions/réponses doit être dérivé des thèmes et faits présents dans les sources et, le cas échéant, du bloc manuel ; ne pas inventer de hors‑sujet.
- QCM : les 3 mauvaises réponses (distracteurs) doivent être plausibles et proches du sujet (erreurs classiques, confusions de cours), pas absurdes.
- OPEN : "answer" au plus 2 phrases, style réponse attendue en oral ou à l’examen court.
- Chaque carte doit avoir une "explanation" pédagogique distincte de la seule reformulation de la réponse.
